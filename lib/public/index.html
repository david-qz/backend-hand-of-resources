Hand of Resoures API

Description

This app hosts a RESTful JSON API with CRUD routes for five different resources:

  • Beverages on /beverages
  • Captains on /captains
  • Inventions on /inventions
  • Planets on /planets
  • Platonic Solids on /platonic-solids

Routes

The following routes exist for each resource:

Route Description
GET /resource Returns with a list of items.
GET /resource/:id Returns with details about item with given id.
POST /resource Adds an item to the database, returning the full item back.
PUT /resource/:id Updates an item in the database, returning the full item back.
DELETE /resource/:id Deletes an item from the database, returning the deleted item back.
